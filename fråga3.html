Fråga 3 - CSS

1. Vad är en regel i CSS och vad består den av?

2. Vad menas med kaskad i CSS?

Svar:

1:

En regel i CSS är en gruppering av en eller flera egenskaper man vill applicera till HTML element. Regeln består av en selektor och en uppsättning egenskaper. Selektorn pekar ut vilken klass eller vilket id man vill applicera regeln på och sedan kan man bestämma bakgrundsfärg, storlek och en nära oändlig mängd egenskaper man vill koppla till det elementet.


2:

Kaskad i avseende till CSS som får del av sitt namn från begreppet (cascading style sheets) avser kort i vilken ordning regler appliceras. Man kan hamna i ett scenario där två regler specifieras för ett element och då väljer CSS att applicera den regel som definieras längst ned i dokumentet. Detta är dock en sanning med viss modifikation som inte appliceras i det fall då element specifieras närmare. Exempelvis ifall vi har en klass på ett h1 element som vi döpt till "heading" och först i kaskadordning specifierat att denna ska vara röd, så kommer efterföljande regler riktade mot enbart h1-element att ignoreras.
